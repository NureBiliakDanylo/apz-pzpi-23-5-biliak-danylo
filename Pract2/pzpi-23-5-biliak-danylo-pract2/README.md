# ChatGPT API Interaction Project (Pract2)

This project is a practical implementation of the architectural principles discussed in the report for Practical Work №2.

## Structure
- `app.py`: Main script for interacting with OpenAI API using streaming and tokenization.
- `requirements.txt`: Python dependencies.
- `.env.example`: Template for environment variables.

## Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Create a `.env` file from `.env.example` and add your `OPENAI_API_KEY`.
3. Run the script: `python app.py`
